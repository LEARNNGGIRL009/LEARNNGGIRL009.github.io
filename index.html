Publications

Research publications and academic contributions

Recent Publications

Classifying tomato root disease through deep learning technique using root images: An adaptive hydroponic system
Taylor & Francis
Utilized deep learning models (ResNet50, ResNet152v2, and modified Inception V3) to predict plant disease using root images rather than leaves. Modified Inception V3 achieved 99.8% accuracy for classifying healthy, E. coli and Salmonella-affected roots, facilitating early disease detection and timely intervention for effective disease control.
Handling Sensor Faults in Hydroponics: A Deep Learning Imputation Technique for Accurate Tomato Yield Prediction
IEEE
Developed Deep Learning Precision Imputation Model (DLPIM) based on GAN architecture to recover missing agricultural data in IoT-based hydroponic systems. The model achieved optimal accuracy (MSE=2.367-2.664, MAE=0.795-0.851, R²=0.990-0.992) across different missing data scenarios (10-60%), establishing a new benchmark for agricultural time series imputation.
Missing value imputation techniques used in deep learning algorithms: A review
Review Paper
Comprehensive review of state-of-art practices for handling missing data in Machine Learning and Deep Learning algorithms. Discusses types of missing value imputation techniques, their positive and negative impacts, and future research possibilities since 60% of an algorithm's efficiency depends on data quality.
